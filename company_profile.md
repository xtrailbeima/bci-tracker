# NeuroWorm — 仿生蠕动式动态微纤维脑机接口

> 中国科学院深圳先进技术研究院（SIAT）跨学科团队研发

## 核心定位

NeuroWorm（仿生蠕动动态电极）代表脑机接口技术的范式转变：从刚性、静止探针向主动、智能、活体装置演进。

## 核心技术优势

- **柔性材料**：杨氏模量 < 3 MPa，拉伸率 ~100%，与软组织弹性匹配
- **高通道密度**：单根纤维集成最多 60 个独立通道，阻抗 < 100 kΩ
- **主动移动**：磁场控制下可在脑组织、微血管（0.5-1mm）和肌肉筋膜间"蠕动"导航
- **长期稳定**：43 周慢性监测稳定信号采集，54 周纤维包裹 < 23 µm（传统刚性电极 > 451 µm）
- **双模态**：同时进行生物电位感知和生物力学监测

## 解决行业痛点

1. **组织损伤**：刚性电极损伤软脑组织（Neuralink 85% 电极失效案例）
2. **靶点漂移**：传统电极植入后无法调整，需二次手术
3. **血管可达性**：突破 Stentrode 只能进入浅表静脉的限制，可达深部微血管
4. **功能单一**：实现录制+刺激同步，超越纯录制设备

## 竞品对比

| 特性 | Stentrode (Synchron) | NeuroWorm |
|------|---------------------|-----------|
| 植入位置 | 浅表静脉窦 | 深部微血管 |
| 功能 | 仅信号记录 | 同时记录+刺激 |
| 材料 | 镍钛合金+铂金 | 柔性磁响应复合物 |
| 精度 | 受限 | 磁场精确导航 |

## 商业规划

- **短期（1-2年）**：技术闭环验证、动物实验、专利布局
- **中期（3-5年）**：医疗器械原型、小规模人体试验、科研产品上线
- **长期（5-10年）**：多中心临床、国际专利合作、产业化量产
